Quer saber o que é um Teclado perfeito?

É o LOGITECH DINOVO EDGE. Não há muitas palavras para classificar este teclado da LOGITECH. O DINOVO EDGE tem um design excelente a roçar na perfeição. Quase que podiamos acabar a análise aqui. Mas não o vamos fazer. O Edge é extremamente fino. Na parte mais larga tem uns impressionantes 1,1 centímetros. Sim, leu bem. Na parte mais fina tem 50 milínetros. É quase uma folha de papel que serve para teclar.


EXCELENTE UTILIZAÇÃO: As teclas têm um curso curto, o que faz com que a tecla seja um processo muito imediato. É um teclado generoso, o que permite que possamos escrever com velocidade sem ter problemas e atropelos de dedos. Existem teclas de atalho para a suspensão do sistema, para maximizar e minimizar a visualizaçãoe para mudar o cursor de posição. Para a multimédia, temos uma tela de acesso direto ao Windows Media Player e um excelente sistema tátil para o volume. Neste, basta deslizar o dedo na vertical para aumnetar o volume ou o contrário para descer. Ao mesmo tempo que o fazemos vemos ser eliminada uma fita de pontos que representa a escala do volume. Um excelente promenor! Para garantir que não tem que ligar um rato, o Edge traz uma pequena área tátil (tipo touchpad dos portáteis) e dois botões do rato. Não podemos dizer que o sistema sirva uma aolicação mais avançadas(Jogar ou trabalhar no Photoshop, por exemplo), mas para o dia-a-dia de trabalho de sistema operativo chega na perfeição. Não existe qualquer cabo entre o teclado e o computador. É tudo por bluetooth via um pequeno emissor USB que deve ligar ao computador. A sincronização é imediata. A Logitech fornece uma docking station onde pode carregar o teclado. No entanto, o consumo é muito baixo, por isso a autonomia é enorme.
